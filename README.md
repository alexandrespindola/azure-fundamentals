# Azure Fundamentals

## Cloud Computing

Cloud computing is the delivery of computing services over the Internet, enabling faster innovation, flexible resources, and economies of scale.

## Cloud Deployment Models

### Public Cloud

- Owned by cloud service providers or hosting companies
- Provides resources and services to multiple organizations and users
- Accessed via secure network connection (typically the Internet)
- No capital expenses for scaling up
- Pay only for what you use

### Private Cloud

- Organizations create a cloud environment in their own data center
- Organizations are responsible for operating the services they provide
- Does not provide access to users outside the organization
- Full control over resources and security
- Responsible for hardware maintenance and updates

### Hybrid Cloud

- Combines public and private clouds
- Allows applications to run in the most appropriate location
- Organizations determine where to run their applications
- Control over security, compliance, and legal requirements
- Provides the greatest flexibility

## Cost Models

### Capital Expenditures (CapEx)

- Initial spending on physical infrastructure
- CapEx expenses have a value that reduces over time

### Operational Expenditures (OpEx)

- Spend on products and services as needed
- Pay-as-you-go model
- Billed immediately for what you use

## Consumption-Based Model

Cloud providers operate on a consumption-based model, where end users pay only for the resources they use. Benefits include:

- Better cost prediction
- Individual resource and service pricing
- Charges based on actual usage
